Ejercicio 29: Crear cuatro input dentro de un formulario , uno para guardar el dni ,otro para guardar el teléfono ,otro para guardar el correo y un tercer botón tipo submit para enviar los datos . Desde el formulario con el evento onsubmit validaremos los datos con la función validar datos . Como vimos ya en los formularios , para llamar a validar pasamos el objeto this como parámetro: validar(this); Recordar que una vez dentro de la function validar(formulario), accederemos a cada uno de los input según el id que hayamos asignado a cada uno de ellos; ejemplo: formulario.dni.value if (formulario.dni.value.length==0) alert(“No ha rellenado el DNI”);
